LOON

A high-level configuration language that compiles down to JSON. Create complex configurations with powerful injection, reusable templates, and intuitive syntax.

Why Choose LOON?

🏷️ Labels & Spaces

Organize your configuration with intuitive hierarchical structure using labels (top-level) and spaces (sub-level) for clear data organization.

🔄 Injection Operator

Reuse and modify configuration data with the powerful injection operator (->). Extract content from labels, spaces, and identities anywhere in your code.

🔗 Dereference & Insertion

Use the dereference operator (&) to inject content only, and the insertion operator (@) to import external LOON files for modular configurations.

📝 Clean Syntax

Write readable configuration files with simple syntax: labels (Label), spaces :space, identities key = value, and comments <like this>.

🎯 JSON Compilation

Compile directly to JSON with Python or JavaScript parsers. Perfect for configuration files, API responses, and structured data.

🔧 Template System

Create reusable templates and override specific values. Build complex configurations from simple, composable pieces.
